# self-introduction
HTTP＆CSSで自己紹介＆オリジナルサイト作成

## 🎨 コンセプト
**「僕らしさを着飾る」ポートフォリオサイト**

見やすさはもちろん、**僕らしく、ワクワクするような感じを醸し出しつつ、華やかに着飾る**ことを目指しています。

### 開発目的
1. より自分のことを分かってもらうため
2. これからの活動により拍車がかかるような自己紹介サイトを作りたい！

### デザイン目標
- 企業のHPにありそうなテイストに仕上げる
- 見る人がワクワクする体験を提供
- 個性的でありながらプロフェッショナルな印象

---

## 🎯 実装済み機能

### ✨ インタラクティブな体験
- **スプラッシュスクリーン**: 初回訪問時のアニメーション演出
- **GSAP アニメーション**: プロフェッショナルなタイムラインアニメーション
  - ヒーローセクション: フェードイン、スケール、スタガー効果
  - スクロール連動: ScrollTriggerで各セクションが滑らかに出現
  - パララックス効果: ヒーロー画像の視差スクロール
  - 弾性アニメーション: AtCoderカードのelastic.out効果
  - 統計カウントアップ: GSAPによる滑らかな数値アニメーション
- **Typed.js タイピング演出**: 
  - 4つのメッセージをループ表示（絵文字付き）
  - カスタムカーソル（▮）のパルスアニメーション
  - グラデーションテキスト＋グローエフェクト
- **スクロールアニメーション**: セクションごとのフェードイン効果
- **ホバーエフェクト**: カードやボタンへの視覚的フィードバック

### 📊 AtCoder レーティング可視化
- **カウントアップアニメーション**: レート値が滑らかに増加 (3秒、easeOutCubic)
- **円形グラフ (SVG)**: 
  - 400点 = 1周回転の視覚化
  - レート色による多層レイヤー表示
  - 12時位置を起点とした時計回り描画
- **スライド＆ポップ演出**: 
  - レートアニメ完了後に左へスライド
  - 参加回数がポップイン表示（scale + fade + blur）
- **目標管理システム**:
  - クリック/キーボード操作で展開可能なゴールカード
  - 瓦("^")による視覚的な達成度表示（100点 = 1瓦）
  - プログレスバーで進捗を一目で確認

## 🚀 これから着飾るための提案

### 🎨 ビジュアル強化ライブラリ

#### 1. **Three.js** - 3D背景・パーティクル効果
- **ワクワク度**: ⭐⭐⭐⭐⭐
- **用途**: ヘッダー背景に3D空間、浮遊するパーティクル、マウス追従エフェクト
- **実装例**: 
  - スキルアイコンが3D空間で回転
  - スクロールに連動した視差効果
  - AtCoderレートに応じた色のパーティクル
```bash
npm install three
```

#### 2. **GSAP (GreenSock Animation Platform)** - プロ級アニメーション
- **ワクワク度**: ⭐⭐⭐⭐⭐
- **用途**: 滑らかで複雑なタイムラインアニメーション
- **実装例**:
  - ページ遷移時のモーフィング効果
  - スクロールトリガーアニメーション
  - 数値カウントアップの強化版
```bash
npm install gsap
```

#### 3. **Lottie** - After Effects アニメーション
- **ワクワク度**: ⭐⭐⭐⭐
- **用途**: デザイナーが作成した高品質アニメーションの再生
- **実装例**:
  - ローディングアニメーション
  - 成功/失敗フィードバック
  - アイコンのマイクロインタラクション
```bash
npm install lottie-web
```

#### 4. **Typed.js** - タイピングエフェクト
- **ワクワク度**: ⭐⭐⭐
- **用途**: テキストが打ち込まれるようなアニメーション
- **実装例**:
  - 自己紹介文のタイピング演出
  - 「趣味: 競技プログラミング▮」のようなカーソル表示
```bash
npm install typed.js
```

#### 5. **AOS (Animate On Scroll)** - スクロールアニメーション簡易化
- **ワクワク度**: ⭐⭐⭐
- **用途**: 既存のスクロールアニメーションをより洗練
- **実装例**:
  - フェード、スライド、ズームなど多様なプリセット
  - data属性で簡単に制御
```bash
npm install aos
```

#### 6. **Chart.js / D3.js** - データ可視化
- **ワクワク度**: ⭐⭐⭐⭐
- **用途**: AtCoderデータのグラフ化
- **実装例**:
  - レート推移の折れ線グラフ
  - コンテスト参加履歴のタイムライン
  - パフォーマンス分布のヒストグラム

## 📝 開発メモ

### アニメーション実装詳細
- **GSAP設定**:
  - ScrollTrigger: `start: 'top 80%'` でビューポート80%位置で発火
  - Stagger: 0.1〜0.3秒で順次表示
  - Easing: `power3.out`, `back.out(1.7)`, `elastic.out(1, 0.5)`
  - パララックス: `scrub: 1` で滑らかな視差効果
- **Typed.js設定**:
  - タイピング速度: 50ms/文字
  - 削除速度: 25ms/文字
  - 表示時間: 2.5秒
  - カーソル: `▮` でパルスアニメーション

### ドキュメント方針
- 全ファイル(index.html, styles.css, script.js)に詳細な構造化ドキュメントヘッダーを記載
- 関数・変数の説明、依存関係を明記
- Last updated日付で更新履歴を追跡

### コーディング規約
- **アニメーション**: 120fps (setInterval 8.33ms) でなめらかに
- **イージング**: easeOutCubic (1 - (1-t)³) を基本とする
- **色管理**: CSS変数で一元管理、レート色は専用配列
- **アクセシビリティ**: ARIA属性、キーボード操作、スクリーンリーダー対応
